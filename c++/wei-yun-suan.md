### 判断一个数的奇偶性

如果一个十进制的数用二进制来表示，那么二进制的最后一位是1表示这个数是奇数，如果这个数的最后一位是0表示这个数是偶数。

```cpp
if(0 == (a & 1))
{
    //a 是偶数
}
```

这一部分在虚幻引擎的源码当中是看到过的。 当写到这块的时候突然想起来了，灵光乍现，这个需要好好记录一下。
